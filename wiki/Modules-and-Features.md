## 1. Dashboard

**Available in:** LE-SOFT, Website Admin, Mobile Staff

The central landing screen after login. Provides an at-a-glance summary of business performance.

**Key widgets (LE-SOFT):**
- Sales summary (today / this month / this year)
- Pending orders and production status
- Stock alerts (low-stock items)
- Recent transactions
- Quick-action shortcuts

---

## 2. CRM — Customer Relationship Management

**Available in:** LE-SOFT, Mobile Staff

Manages all customers, dealers, and prospects.

| Feature | Description |
|---------|-------------|
| CRM Directory | Full contact list of customers with search and filter |
| CRM Progress | Track customer interaction history and pipeline stage |
| Customer Ledger (List) | All customers with outstanding balance summary |
| Customer Ledger (Detail) | Full transaction history for a specific customer |
| Exchange Orders | Create and manage product exchange/return orders |

---

## 3. Quotations

**Available in:** LE-SOFT, Mobile Staff

Manages the pre-sales quoting process.

| Feature | Description |
|---------|-------------|
| Quotation Create | Build a detailed quotation with line items and pricing |
| Quotation List | Browse, search, and manage all quotations |
| Quotation Preview | Printable/PDF preview of a quotation |

---

## 4. Billing / Invoicing

**Available in:** LE-SOFT, Mobile Staff

Handles sales invoices and billing adjustments.

| Feature | Description |
|---------|-------------|
| Billing | Create new sales invoices |
| Bill History | Browse and search historical invoices |
| Alter Bill | Modify an existing bill (price changes, quantity corrections) |
| Pending Alterations | View bills awaiting alteration approval |

---

## 5. Make — Production / Workshop

**Available in:** LE-SOFT, Mobile Staff

Manages the furniture production workflow.

| Feature | Description |
|---------|-------------|
| Make Dashboard | Overview of production status across all orders |
| Place Order | Create a new production/work order |
| Track Orders | Monitor in-progress and completed production orders |
| Alter Order | Modify an existing production order |

---

## 6. Inventory / Stock

**Available in:** LE-SOFT, Mobile Staff

Manages raw materials, semi-finished goods, and finished product inventory.

**Stock Masters:**
| Feature | Description |
|---------|-------------|
| Product Create / List | Finished product catalog (for sale) |
| Stock Group Create / List | Categorize stock items into logical groups |
| Stock Item Create / List | Raw materials and components |
| Unit Create / List | Define units of measurement (pcs, kg, m, etc.) |

## 7. Shipping / Delivery

**Available in:** LE-SOFT, Mobile Staff

| Feature | Description |
|---------|-------------|

## 8. Accounting

**Available in:** LE-SOFT, Mobile Staff

Full double-entry accounting system.

**Accounting Masters:**
| Feature | Description |
|---------|-------------|
| Group Create / List | Chart-of-accounts group hierarchy |
| Ledger Create / List | Individual ledger accounts (assets, liabilities, income, expenses) |

**Vouchers:**
| Feature | Description |
|---------|-------------|
| Voucher Entry | Create journal/payment/receipt vouchers |
| Voucher List | Browse all accounting vouchers |

## 15. Settings

**Available in:** LE-SOFT, Website Admin, Mobile Apps

| Setting | Description |
|---------|-------------|
| Company Profile | Business name, logo, address, contact details |

| Device Monitoring | Enable/disable local SQLite backup |
| Auto-Update | Check for and install software updates |
| License | View Machine ID and license status |
| Security | Change password, session management |
